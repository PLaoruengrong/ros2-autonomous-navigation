# ROS2 Autonomous Navigation

This repository contains introductory autonomous navigation projects developed using ROS2 simulation environments.

The projects explore robot motion control, sensor-based navigation, and obstacle avoidance behaviors through custom controller implementations.

## Features

* Twist-based robot motion control
* Sensor-driven obstacle avoidance
* Autonomous path-following behavior
* ROS2 simulation workflow

## Project Demonstrations

### 1. Twist-Based Navigation

Robot movement controlled using Twist command inputs for directional navigation and velocity control.

### 2. Sensor-Based Autonomous Navigation

Autonomous obstacle avoidance using simulated sensor feedback and reactive control logic.

## Technologies Used

* ROS2
* C++
* Robot simulation environment
* Differential drive control
* Sensor feedback systems

## Repository Structure

* `controllers/` — robot controller implementations
* `videos/` — simulation demonstrations
* `figures/` — screenshots from simulation environment

## Learning Objectives

* Understanding ROS2 communication workflows
* Implementing autonomous navigation logic
* Applying sensor feedback to robot control
* Developing modular robotics controllers
